# ESPHome Dual Blinds Controller (ULN2003)

Управление двумя шторами/жалюзи на базе ESP32/ESP8266 и двух шаговых двигателей 28BYJ-48 с драйверами ULN2003.

## Возможности
- 🪟 Независимое управление каждой шторой
- 🪟 Виртуальная штора "Обе" с усреднением позиций
- 💾 Сохранение позиций во flash-память (переживает перезагрузку)
- 🎯 Калибровка нулевой позиции одной кнопкой
- ⏱️ Автосохранение текущей позиции при достижении цели

## Аппаратная часть
- **Платы**: ESP32 DevKit **или** Wemos D1 Mini (ESP8266)
- **Драйверы**: 2x ULN2003
- **Двигатели**: 2x 28BYJ-48 (5V или 12V)
- **Питание**: 12V 2A с преобразователем в 3.3V (рекомендуется)

> ⚠️ **Важно**: Не питайте шаговые двигатели от пина 5V/3.3V ESP! Используйте отдельный блок питания 12V 2A+ с преобразователем и общей землёй.
